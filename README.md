# License Key Reader

![Main Window](res/screenshot.png?raw=true "License Key Reader")

## Install

```sh
npm install
```

## Run 

```sh
npm start
```

## Known issues

If the settings.json file is missing from the application root folder, it does not create one, and selecting new file locations is not possible.
